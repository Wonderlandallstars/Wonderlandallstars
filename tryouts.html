ELITE TRYOUT REQUIREMENTS

General Tryout Format ▼

Athletes will perform: • 2 Standing Tumbling Passes (chosen from the list) • 2 Running Tumbling Passes (chosen from the list) • 3 Jumps (at least 2 different types)

Jumps Required:

  • Toe Touch
  • Hurdler (Left or Right)
  • Pike

Prep & Novice ▼

  • Back Walkover
  • Front Walkover
  • Cartwheel
  • Round Off
  • Forward Rolls
  • Backward Rolls

Level 1 ▼

  • Standing 3 Back Walkovers
  • Valdez → 2 Back Walkovers
  • Cartwheel → 2 Back Walkovers
  • Front Walkover → Cartwheel → Back Walkover
  • Front Walkover → Round Off → 2 Back Walkovers
  • Round Off → Back Walkover → Switch Leg → Back Walkover

Level 2 ▼

  • Back Walkover → Back Handspring
  • Back Handspring Step-Out → Back Walkover → Back Handspring
  • Back Extension Roll → Back Walkover → Back Handsprings
  • Round Off → 3 Back Handsprings
  • Front Walkover → Round Off → 2 Back Handsprings
  • Flyspring / Bounder

Level 3 ▼

  • Standing 3 Back Handsprings
  • Jump → 2 Back Handsprings
  • Back Handspring → Jump → Back Handspring
  • Aerial
  • Front Walkover → Aerial
  • Round Off → Back Handspring → Tuck
  • Front Walkover → Round Off → Back Handspring → Tuck

Level 4 ▼

  • Standing Back Tuck
  • Back Handspring → Back Tuck
  • Jump → Back Handspring → Tuck
  • Punch Front → Step Out → Round Off → Back Handspring → Layout
  • Round Off → Back Handspring → Layout
  • Round Off → Whip → Back Handspring Series → Layout

Level 5 ▼

  • Back Handspring → Whip → Back Handspring Series → Layout
  • Jump → Back Handspring Series → Layout
  • Back Spring → Whip → Back Spring Series → Layout
  • Front Walkover → Round Off → Back Handspring → Full
  • Round Off → Whip → Back Handspring → Full
  • Punch Front → Step Out → Round Off → Back Handspring → Full

Level 6 ▼

  • Jump → Back Tuck
  • Back Spring Series → Full
  • Standing Full
  • Round Off → Arabian → Round Off → Back Handspring → Full
  • Round Off → Double Full
  • Round Off → Back Handspring → Full → Back Handspring Series → Full
